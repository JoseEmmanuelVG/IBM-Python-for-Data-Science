## ðŸ§µ String Operations

### String Creation and Indexing

```python
name = "Lizz"
print(name[0:2])  # â†’ 'Li'
```

- `name[0]` â†’ `'L'`
- `name[1]` â†’ `'i'`
- `name[0:2]` â†’ `'Li'` (up to but not including index 2)

---

### String Slicing with Step

```python
var = "01234567"
print(var[::2])   # â†’ '0246'
```

This takes every second character, starting from index 0.

---

### String Concatenation

```python
'1' + '2'  # â†’ '12'
```

This merges both strings together.
