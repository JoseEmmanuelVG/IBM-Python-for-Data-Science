```python

```
Each fruit name is printed in uppercase:

- 'APPLE'
- 'BANANA'

âœ… Output:

#### `while` loop example:
```python

    print(f"Count: {counter}")
    counter += 1
```
This loop prints values from `counter = 0` to `counter = 2`

âœ… Output:
